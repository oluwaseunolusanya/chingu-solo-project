# chingu-solo-project                [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description                       
Project employs object-oriented programming to implement an onboarding command-line app used to create new employee profile. User is prompted to enter new hire name, role and start date. The program exits and prints how many employee profiles were created after the user indicates no more employee to be added.

## Installation 
1. Open your terminal.
2. Check that git and python are installed on your local machine. For example, entering _python --version_ at the prompt will return version of python if installed. Similarly for _git --version_.
```bash
python --version
```
3. Perform there installation if not installed - See guides at _https://www.python.org/downloads/_ for python and _https://git-scm.com/_ for git.
4. Navigate from the current working directory to the directory of choice or to a newly created directory if need be. For example,
```bash
 mkdir python-projects
```
5. Enter git clone https://github.com/oluwaseunolusanya/chingu-solo-project.git at the prompt to create a local copy of the repo.
```bash
git clone https://github.com/oluwaseunolusanya/chingu-solo-project.git 
```
7. Type ls -al content of the repo.
```bash
ls -al
```
